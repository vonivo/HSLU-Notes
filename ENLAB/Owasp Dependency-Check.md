Owasp Dependency-Check ist ein automatisiertes Tool, welches genutzt wird um die Codebase und die Dependencies auf bekannte Vulnerabilities zu untersuchen.


>[!Info]
>A vulnerability is a hole or a weakness in the application which can be a desing flaw or an implementation bug, that allows an attacker to cause harm to the stakeholder of an application. Stakeholder include the application owner, application users and other entities that rely on the application.

Owasp nutzt die National Vulnerability Database des NIST und es können auch ausnahmen definiert werden.
