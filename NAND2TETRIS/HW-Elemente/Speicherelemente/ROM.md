>[!Definition]
>Der Hack Befehlsspeicher ist in einem Nur-Lese-Speicher, auch ROM genannt, implementiert. Der Hack-Rom besteht aus 32K adressierbare 16-Bit Registern.


|               |                   |
| ------------- | ----------------- |
| **Chipnanme** | ROM               |
| **Eingänge**  | `address[15]`     |
| **Ausgägne**  | `out[16]`         |
| **Funktion**  | `out=ROM[addres]` |
