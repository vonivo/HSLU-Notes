Das Speichern von Informationen ist immer zeitabhängig.
Damit die Zeit definiert ist, besitzen die meisten Computer einen Haupttaktgeber, welcher eine kontinuierliche Folge von Wechselsignalen abgibt.
![[Pasted image 20260615162945.png]]

Um Daten zu speicher werden neben kombinatorischen Chips auch Sequentielle, an einen Takt gebundene Chips benötigt.
![[Pasted image 20260615163054.png]]
>[!Info]
>Die **Diskretisierung** der Ausgaben der sequentiellen Chips kann zur Synchronisierung der gesamten Computerarchitektur verwendet werden.
