| **Chipnanme** | Screen                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Eingägne**  | `in[16]`<br>`load`<br>`address[13]`                                                                                                                                                                                                                                                                                                                                                                                |
| **Ausgäange** | `out[16]`                                                                                                                                                                                                                                                                                                                                                                                                          |
| **Funktion**  | Funktioniert genau wie ein 16-Bit 8K-RAM.<br>Wenn `load=1` dann `Screen[address(t-1)](t) = in(t-1)`<br>`out(t)=Screen[address(t)](t)`                                                                                                                                                                                                                                                                              |
| **Anmerkung** | Hat den Nebeneffekt, dasse in 256\*512 Schwarz Weiss Bildschirm ständig aufgefrischt wird. Jede Zeile des Bildschrims wird durch 32 aufeinanderfolgende 16-Bit-Worte dargestellt, beginnen in der linken oberen Ecke. Das Pixel in der Zeile `r` von oben der Spalte `c` von links ($0\leq r\leq 255, 0\leq c\leq 511$) spieget das $c\%16-Bit$ (von LSB zu MSB gezähtl) des Wortes in `SCREEN[r*32+c/16]` wieder. |
