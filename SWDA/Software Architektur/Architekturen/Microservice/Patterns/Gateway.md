>[!Definition]
>Das Gateway entspricht dem Konzept des Fassaden-Patterns (GoF) auf Ebene der Architektur. Er vereinfacht die Implementation des Clients deutlich

Der Gateway übernimmt dabei auch gewisse Aufgaben:
- Authentifizierung und Verschlüsselung
- Einfache Mappings / Transfomationen von Daten.

Ein Gateway kann als expliziter Service implementiert werden oder es können auch generische Alternativen genutzt werden.
